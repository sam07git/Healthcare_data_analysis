# Healthcare_data_analysis
## Healthcare Data Analysis using Power BI

### Introduction

Developed an interactive Power BI dashboard to analyze hospital visitor data and uncover trends in patient demographics, service usage, wait times, and satisfaction levels. Used Power Query for data transformation, DAX for calculations, and data modeling techniques to create meaningful business insights that support operational efficiency and improved patient care.

### Technologies Used

* Power BI
* Power Query Editor
* DAX (Data Analysis Expressions)
* Excel

### Key Features

**Data Preparation**

* Performed ETL operations using Power Query.
* Cleaned missing values and transformed data.
* Split data into multiple related tables and established relationships for better analysis.

**Data Modeling & DAX**

* Created calculated measures and KPIs using DAX.
* Built relationships between tables to enable interactive reporting and filtering.

**Insights Generated**

* Analyzed daily, monthly, quarterly, and yearly visitor trends.
* Studied patient demographics including age, gender, and race.
* Evaluated wait times and visitor satisfaction scores.
* Examined department referrals and patient registration status.
* Identified patterns in weekday/weekend and AM/PM visits.

### Business Impact

The dashboard helps healthcare professionals monitor patient flow, understand visitor behavior, optimize resource allocation, reduce wait times, and improve overall patient satisfaction through data-driven decision-making.
